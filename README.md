# Stock Market Expense Tracker

A simple, single-file web app to track every rupee you spend on the stock market — losses, brokerage, taxes, courses, webinars, subscriptions, and more.

## Features

- Tracks 12 expense categories grouped into Losses, Charges, Education, and Profits
- Net position calculated automatically
- Visual breakdown bar chart by category
- Filter entries by group
- Export and import CSV for backup
- Data saved in your browser (localStorage)
- Works offline — no server, no signup
- Auto dark mode based on system preference
- Mobile responsive

## How to host on GitHub Pages

1. Create a new repository on GitHub (e.g. `stock-tracker`)
2. Upload `index.html` to the root of the repo
3. Go to repo Settings → Pages
4. Under "Build and deployment", set Source to "Deploy from a branch"
5. Choose branch `main` and folder `/ (root)`, then Save
6. Wait a minute, then visit `https://YOUR-USERNAME.github.io/stock-tracker/`

Done. Your tracker is live.

## How to use locally

Just open `index.html` in any browser. That's it.

## Backup tip

Your data lives in browser localStorage. If you clear browser data or switch devices, it's gone. Use the **Export CSV** button regularly to back up. You can re-import the CSV later.

## Privacy

Everything stays on your device. No analytics, no tracking, no data sent anywhere.
